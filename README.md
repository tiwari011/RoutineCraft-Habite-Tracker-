RoutineCraft – Habit Tracker Web App

RoutineCraft is a full-stack Habit Tracker web application that helps users build and maintain daily routines efficiently. Users can securely manage their habits, set completion times, and track progress with automatic daily reset functionality.

🚀 Features
🔐 Authentication

User Signup & Login

JWT-based authentication

Protected routes using middleware

Secure API access with token authorization

📝 Habit Management
➕ Add Habits

Create new habits

Set a specific time for completion (e.g., 7:00 AM, 9:30 PM)

Habits are stored in the database

✏️ Edit Habits

Update habit name

Modify completion time

Changes reflect instantly

❌ Delete Habits

Remove habits permanently

Data is deleted from the database

✅ Habit Completion Tracking

Mark habits as completed

Toggle completion status

Visual indication of completed habits

⏳ Daily Auto Reset

All completed habits automatically reset after 24 hours

Ensures fresh tracking every new day

Maintains consistency in daily routine building

Frontend

Backend

Express.js

Mongoose

JWT Authentication Middleware
